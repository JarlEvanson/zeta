# Zeta

### What is Zeta?
Zeta is a operating system written in Rust. Zeta, following the seL4 microkernel design, 
aims to be reliable, secure, usable, correct, and free.

Zeta aims to be *not* just a kernel, it aims to be a fully featured operating system, providing
components like a memory allocator, file system, display manager, etc., that make up an operating
system.

### Supported Architectures
* x86-64 (UEFI)
